📖 Project Description

Root & Bloom is a modern and responsive plant nursery website designed to provide users with an attractive platform for exploring a variety of indoor and outdoor plants. The website showcases popular plants, their prices, nursery services, company information, and contact details through a clean and user-friendly interface.

The project was developed as part of a Responsive Web Layout assignment, focusing on creating a website that adapts seamlessly across desktop, tablet, and mobile devices using modern CSS techniques.

🎯 About Responsive Layout

A responsive layout ensures that a website automatically adjusts its structure and appearance according to different screen sizes and devices.

In this project:

Desktop view displays multiple plant cards in a grid layout.
Tablet view adjusts the layout to fewer columns.
Mobile view displays content in a single-column format.
Navigation and content remain accessible on all devices.

This responsiveness is achieved using CSS Grid, Flexbox, and Media Queries.
